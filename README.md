# Electric Grid Disruption Analysis

This project uses Python-based descriptive analytics to examine U.S. electric grid disruptions from 2000 to 2014.

---

## Project Overview

The goal of this project is to identify patterns in grid disruptions, including:

- Common causes of outages  
- Trends over time  
- Regional differences (NERC regions)  
- Impact on customers  

This analysis is especially relevant to system operations in the El Paso and Las Cruces area, where El Paso Electric operates within the WECC region rather than ERCOT.

---

## Tools & Technologies

- Python  
- Pandas  
- Matplotlib  
- Jupyter Notebook  

---

## Key Insights

- Vandalism and physical attacks are among the most common causes of disruptions  
- Outage frequency varies significantly over time, with a spike around 2011–2012  
- WECC and RFC regions experience higher disruption counts  
- Most outages affect a small number of customers, but a few large events create significant impact
